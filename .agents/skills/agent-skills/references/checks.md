# Understanding Checks: Agent Skills

## Check Questions
1. What is an agent skill, and how is it different from a code library or API?
2. What does AGENTS.md do in a repo? How does it shape agent behavior?
3. Look at the SKILL.md for this topic. What are the key sections and why does each exist?
4. How does this learning experience use repo-level context engineering? Walk through the flow from AGENTS.md to skill to concept file.
5. If you were building a repo that agents would work in, what would you put in the AGENTS.md?

## Completion Criteria
- The learner can explain the role of AGENTS.md as an entry point for agent behavior
- They understand skills as packaged capabilities -- instructions, context, and process
- They can articulate how this curriculum repo uses context engineering to create the learning experience
- They see the connection between repo structure and agent effectiveness

## What to Update
- `learner/progress.md` -- mark "Agent Skills" complete
- `learner/notes.md` -- record their meta-awareness level, any insights about the learning experience design
