# Learning Resources

Curated external resources for AI engineering. Quality over quantity — only what's genuinely valuable.

---

## Foundations

### Understanding LLMs
- **"Intro to Large Language Models" by Andrej Karpathy** (Video) — The best single explanation of what LLMs are and how they work. Start here.
- **"Attention Is All You Need" paper** (Paper) — The original transformer paper. You don't need to understand the math, but knowing the architecture helps.
- **"State of GPT" by Andrej Karpathy** (Video) — How GPT models are trained, from pre-training to RLHF.
- **Anthropic's research blog** (Blog) — Deep technical content on model behavior, safety, and capabilities.

### Python for AI
- **Official Python documentation** (Docs) — The authoritative reference for Python features.
- **Real Python** (Website) — Practical tutorials on Python patterns used in AI development.
- **FastAPI official tutorial** (Docs) — Excellent, comprehensive guide to building APIs with FastAPI.

## Core Skills

### Prompt Engineering
- **Anthropic's prompt engineering guide** (Docs) — Official guidance on prompting Claude effectively. Applicable to all models.
- **OpenAI's prompt engineering guide** (Docs) — OpenAI's best practices for prompting.
- **"Prompt Engineering Guide" by DAIR.AI** (Website) — Comprehensive open-source guide to prompt techniques.

### RAG and Embeddings
- **LangChain documentation** (Docs) — The primary reference for building RAG pipelines with LangChain.
- **Qdrant documentation** (Docs) — Official docs for the recommended vector database.
- **"Building RAG Applications" by LangChain** (Tutorial) — Step-by-step guide to RAG with LangChain.

### Tool Calling
- **OpenAI function calling guide** (Docs) — Clear explanation of the tool calling pattern.
- **Anthropic tool use documentation** (Docs) — Claude's approach to tool calling.
- **LangChain tools documentation** (Docs) — Building tools with LangChain.

## Agent Architecture

### Agents and Frameworks
- **LangChain documentation** (Docs) — The primary framework for building agents. Start with the agents section.
- **LangGraph documentation** (Docs) — LangChain's graph-based agent framework for complex workflows.
- **"What is an AI Agent?" by Harrison Chase** (Blog) — LangChain founder's overview of agent architecture.

### MCP
- **Model Context Protocol specification** (Docs) — The official MCP documentation.
- **MCP server examples** (GitHub) — Reference implementations for building MCP servers.
- **Anthropic's MCP blog post** (Blog) — Introduction to MCP and why it matters.

### Context Engineering
- **"Context Engineering" by Tobi Lutke** (Post) — Shopify CEO's framing of context engineering as the new skill.
